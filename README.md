
# ShoppyGlobe API (Node.js + Express + MongoDB)

A clean, modern, fully ESM backend for the ShoppyGlobe e-commerce app.

git repo - https://github.com/rajesh36sarkar/shoppyglobe-api

## Features
- Products API (CRUD)
- Cart API (Add/Update/Delete)
- JWT Authentication (Register/Login)
- MongoDB with Mongoose
- Clean folder structure
- Fully formatted using Prettier

## Folder Structure
```
/src
  /config
  /controllers
  /middlewares
  /models
  /routes
server.js
```

## Installation
```
npm install
npm start
```
